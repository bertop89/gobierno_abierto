# Portal de Votaciones Parlamentarias

Este proyecto es un portal web que muestra información sobre votaciones parlamentarias, permitiendo a los usuarios consultar y analizar los resultados de diferentes votaciones.

## Características

- Visualización de votaciones parlamentarias
- Detalles de cada votación incluyendo:
  - Fecha
  - Título
  - Descripción
  - Resultados (Sí, No, Abstenciones)
  - Gráficos de resultados
- Búsqueda de votaciones
- Vista detallada de cada votación
- Diseño responsive

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON para almacenamiento de datos

## Estructura del Proyecto

```
.
├── assets/
│   └── styles.css
├── data/
│   └── *.json
├── index.html
├── detalle.html
├── acerca.html
└── script.js
```

## Instalación

1. Clonar el repositorio
2. Abrir `index.html` en un navegador web

## Uso

1. Navega a la página principal
2. Utiliza la barra de búsqueda para filtrar votaciones
3. Haz clic en cualquier votación para ver sus detalles
4. En la vista de detalle, puedes:
   - Ver información completa de la votación
   - Filtrar votaciones individuales
   - Ordenar la tabla por diferentes columnas
