Sobre Votaciones Parlamentarias

Esta aplicación web permite visualizar y analizar las votaciones realizadas en el Parlamento de manera clara y accesible. Diseñada para proporcionar transparencia en el proceso democrático, permite a los ciudadanos:

  • Consultar los resultados de las votaciones parlamentarias
  • Ver el detalle de cada votación, incluyendo los votos individuales
  • Buscar votaciones específicas por texto
  • Filtrar los resultados por diputado o grupo parlamentario

Características Técnicas

La aplicación ha sido desarrollada utilizando tecnologías web modernas:

  • HTML5 y CSS3 para la estructura y el diseño
  • JavaScript vanilla para la funcionalidad interactiva
  • Diseño responsivo que se adapta a diferentes dispositivos
  • Arquitectura basada en archivos JSON para el almacenamiento de datos

Visualización de Datos

Los resultados se presentan de forma visual mediante:

  • Barras de progreso para mostrar la proporción de votos
  • Tablas interactivas con capacidad de ordenamiento
  • Indicadores visuales para diferentes tipos de votos
  • Sistema de paginación para manejar grandes conjuntos de datos
